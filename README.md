## Getting Started

### Prerequisites

* Node.js & npm
* Expo CLI (`npm install -g expo-cli`)

### Installation

```bash
npm install
```

### Running the App

```bash
npx expo start
```

Then scan the QR code using Expo Go on your mobile device or run in an emulator.
